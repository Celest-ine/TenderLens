# TenderLens
**TenderLens** is an AI-powered tender document analysis platform that transforms Request for Tender(RFT) and Request for Proposal(RFP) documents into structured, categorized complience checklists.

Designed for tender consultants, procurement professionals, and businesses, TenderLens reduces manual document review, improves compliance, and helps users prepare competitive tender submissions more efficiently.

## Project Overview

### Problem Statement
- Tender documents are often lengthy and complex, with legal, administrative, technical and financial requirements spread across multiple sections.

- Manual review is time-consuming, prone to human error and increases the risk of missed requirements, poor bid/no bid decisions and unsuccessful submissions.

---

### Solution
TenderLens automates the first stage of tender analysis by extracting and organizing key requirements into a clear, structured checklist.

## Features

### Planned

- Upload tender documents (PDF)

- Identify document sections
